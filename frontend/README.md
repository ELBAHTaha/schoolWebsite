# Frontend (React) for JEFAL Privé

This folder contains the React/CSS frontend used as a reference while the Laravel backend UI is being built. It is optional for production but useful for prototyping and design.

## Run locally

```bash
cd frontend
npm install
npm run dev
```

The dev server runs at http://localhost:5173 by default.

## Notes

- The frontend is intended as a reference UI; the final production build will be served from the Laravel backend (`backend/`).
- If you change dependencies, run `npm install` again to update `package-lock.json`.
